What is Sublist3r?

Sublist3r is a fast subdomain enumeration tool for penetration testers. It enumerates subdomains using many search engines such as Google, Yahoo, Bing, Baidu, and Ask. Sublist3r also enumerates subdomains using Netcraft, Virustotal, ThreatCrowd, DNSdumpster, and ReverseDNS. The tool helps security professionals and bug bounty hunters discover subdomains that could potentially be vulnerable or expose additional attack surfaces. It's written in Python and designed to be fast, efficient, and easy to use.

Automation and Scripting

Batch Processing

# Multi-domain enumeration script
#!/bin/bash
domains=("example.com" "test.com" "demo.com")
output_dir="subdomain_results"
mkdir -p "$output_dir"

for domain in "${domains[@]}"; do
    echo "Enumerating subdomains for $domain"
    python sublist3r.py -d "$domain" -b -v -o "$output_dir/${domain}_subdomains.txt"
    sleep 5
done

# Combine all results
cat "$output_dir"/*_subdomains.txt | sort -u > "$output_dir/all_subdomains.txt"

Result Processing

# Python script for processing results
import re
import requests

def validate_subdomains(filename):
    with open(filename, 'r') as f:
        subdomains = f.read().splitlines()
    
    valid_subdomains = []
    for subdomain in subdomains:
        try:
            response = requests.get(f"http://{subdomain}", timeout=5)
            if response.status_code == 200:
                valid_subdomains.append(subdomain)
                print(f"Valid: {subdomain}")
        except:
            pass
    
    return valid_subdomains

# Usage
valid = validate_subdomains('subdomains.txt')
with open('valid_subdomains.txt', 'w') as f:
    for subdomain in valid:
        f.write(f"{subdomain}\n")

Continuous Monitoring

# Continuous subdomain monitoring script
#!/bin/bash
domain="example.com"
baseline="baseline_subdomains.txt"
current="current_subdomains.txt"

# Initial baseline
if [ ! -f "$baseline" ]; then
    python sublist3r.py -d "$domain" -b -o "$baseline"
fi

while true; do
    echo "Checking for new subdomains..."
    python sublist3r.py -d "$domain" -b -o "$current"
    
    # Find new subdomains
    new_subdomains=$(comm -13 <(sort "$baseline") <(sort "$current"))
    
    if [ ! -z "$new_subdomains" ]; then
        echo "New subdomains found:"
        echo "$new_subdomains"
        # Send alert or notification
        echo "$new_subdomains" | mail -s "New subdomains for $domain" admin@example.com
        
        # Update baseline
        cp "$current" "$baseline"
    fi
    
    sleep 3600  # Check every hour
done

# Custom script with VirusTotal API
import requests
import json

def get_subdomains_vt(domain, api_key):
    url = f"https://www.virustotal.com/vtapi/v2/domain/report"
    params = {'apikey': api_key, 'domain': domain}
    response = requests.get(url, params=params)
    
    if response.status_code == 200:
        data = response.json()
        if 'subdomains' in data:
            return data['subdomains']
    return []

Certificate Transparency

# Query Certificate Transparency logs
import requests
import json

def get_ct_subdomains(domain):
    url = f"https://crt.sh/?q=%.{domain}&output=json"
    response = requests.get(url)
    
    if response.status_code == 200:
        certificates = response.json()
        subdomains = set()
        
        for cert in certificates:
            name = cert.get('name_value', '')
            if name:
                subdomains.update(name.split('\n'))
        
        return list(subdomains)
    return []
