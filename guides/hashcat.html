What is Hashcat?

Hashcat is the world's fastest and most advanced password recovery utility, supporting five unique modes of attack for over 300 highly-optimized hashing algorithms. It utilizes GPU acceleration to achieve incredible cracking speeds, making it the go-to tool for password security testing and recovery.

Basic Usage & Syntax

Command Structure

# Basic syntax
hashcat [options] [hash_file] [wordlist]

# With hash mode
hashcat -m [hash_type] [hash_file] [wordlist]

# Example: Crack MD5 hash
hashcat -m 0 hash.txt rockyou.txt

Common Hash Types

# MD5
-m 0

# SHA1
-m 100

# SHA256
-m 1400

# NTLM (Windows)
-m 1000

# bcrypt
-m 3200

# WPA/WPA2
-m 2500

# List all supported hash types
hashcat --help | grep -E "^\s+[0-9]+"

Attack Modes

Dictionary Attack (-a 0)

# Basic dictionary attack
hashcat -m 0 -a 0 hash.txt wordlist.txt

# Multiple wordlists
hashcat -m 0 -a 0 hash.txt wordlist1.txt wordlist2.txt

# With rules
hashcat -m 0 -a 0 hash.txt wordlist.txt -r rules/best64.rule

Combination Attack (-a 1)

# Combine two wordlists
hashcat -m 0 -a 1 hash.txt wordlist1.txt wordlist2.txt

# Example: Combine names with numbers
hashcat -m 0 -a 1 hash.txt names.txt numbers.txt

Brute Force Attack (-a 3)

# Brute force with mask
hashcat -m 0 -a 3 hash.txt ?d?d?d?d?d?d

# Custom character sets
hashcat -m 0 -a 3 hash.txt -1 ?l?u?d ?1?1?1?1?1?1?1?1

# Mask attack with increment
hashcat -m 0 -a 3 hash.txt --increment --increment-min=4 --increment-max=8 ?a?a?a?a?a?a?a?a

Hybrid Attacks (-a 6, -a 7)

# Hybrid wordlist + mask
hashcat -m 0 -a 6 hash.txt wordlist.txt ?d?d?d?d

# Hybrid mask + wordlist
hashcat -m 0 -a 7 hash.txt ?d?d?d?d wordlist.txt

Masks & Character Sets

Built-in Character Sets

# ?l = lowercase letters (a-z)
# ?u = uppercase letters (A-Z)
# ?d = digits (0-9)
# ?h = hex digits (0-9, a-f)
# ?H = hex digits (0-9, A-F)
# ?s = special characters
# ?a = all printable ASCII characters

# Examples
?l?l?l?l?l?l?l?l    # 8 lowercase letters
?u?l?l?l?l?l?d?d    # Uppercase + 5 lowercase + 2 digits
?a?a?a?a?a?a        # 6 any printable characters

Custom Character Sets

# Define custom charset
hashcat -m 0 -a 3 hash.txt -1 ?l?u -2 ?d?s ?1?1?1?1?2?2

# Multiple custom charsets
hashcat -m 0 -a 3 hash.txt -1 abcdef -2 123456 -3 !@#$ ?1?2?3?1?2?3

# From file
hashcat -m 0 -a 3 hash.txt -1 charset.txt ?1?1?1?1?1?1

Rules Engine

Using Rules

# Apply rules to wordlist
hashcat -m 0 -a 0 hash.txt wordlist.txt -r rules/best64.rule

# Multiple rule files
hashcat -m 0 -a 0 hash.txt wordlist.txt -r rules/best64.rule -r rules/toggles1.rule

# Generate rules on the fly
hashcat -m 0 -a 0 hash.txt wordlist.txt -j '$1' -k '$2'

Common Rule Functions

# : = do nothing
# l = lowercase all
# u = uppercase all
# c = capitalize first letter
# C = lowercase first, uppercase rest
# t = toggle case
# r = reverse
# d = duplicate
# $ = append character
# ^ = prepend character
# [ = delete first character
# ] = delete last character

# Examples
# $1 = append "1"
# ^@ = prepend "@"
# c $1 $2 $3 = capitalize and append "123"

Performance Optimization

GPU Optimization

# Force specific OpenCL device
hashcat -m 0 -a 0 hash.txt wordlist.txt -d 1

# Workload tuning (1-4, higher = more GPU usage)
hashcat -m 0 -a 0 hash.txt wordlist.txt -w 3

# Optimize for specific GPU
hashcat -m 0 -a 0 hash.txt wordlist.txt -O

# Show performance info
hashcat -m 0 -a 0 hash.txt wordlist.txt --status --status-timer=10

Session Management

# Create session
hashcat -m 0 -a 0 hash.txt wordlist.txt --session=mysession

# Restore session
hashcat --restore --session=mysession

# Remove session
hashcat --session=mysession --remove

Advanced Techniques

Potfile Management

# Show cracked passwords
hashcat -m 0 hash.txt --show

# Custom potfile location
hashcat -m 0 -a 0 hash.txt wordlist.txt --potfile-path=custom.potfile

# Disable potfile
hashcat -m 0 -a 0 hash.txt wordlist.txt --potfile-disable

Distributed Cracking

# Skip to specific position
hashcat -m 0 -a 0 hash.txt wordlist.txt -s 1000000

# Limit number of candidates
hashcat -m 0 -a 0 hash.txt wordlist.txt -l 500000

# Keyspace calculation
hashcat -m 0 -a 3 --keyspace ?a?a?a?a?a?a

Output Options

# Output to file
hashcat -m 0 -a 0 hash.txt wordlist.txt -o cracked.txt

# Output format
hashcat -m 0 -a 0 hash.txt wordlist.txt -o cracked.txt --outfile-format=2

# Show progress
hashcat -m 0 -a 0 hash.txt wordlist.txt --status --status-timer=5

Specific Attack Examples

Windows NTLM Hashes

# Basic NTLM crack
hashcat -m 1000 -a 0 ntlm_hashes.txt rockyou.txt

# With rules
hashcat -m 1000 -a 0 ntlm_hashes.txt rockyou.txt -r rules/best64.rule

# Brute force 8 chars
hashcat -m 1000 -a 3 ntlm_hashes.txt ?a?a?a?a?a?a?a?a

WPA/WPA2 Handshakes

# Convert cap to hccapx first
cap2hccapx capture.cap handshake.hccapx

# Crack WPA2
hashcat -m 2500 -a 0 handshake.hccapx rockyou.txt

# With mask for known pattern
hashcat -m 2500 -a 3 handshake.hccapx ?d?d?d?d?d?d?d?d

Linux Shadow Hashes

# SHA512crypt (most common)
hashcat -m 1800 -a 0 shadow_hashes.txt rockyou.txt

# MD5crypt
hashcat -m 500 -a 0 shadow_hashes.txt rockyou.txt

Pro Tips & Best Practices

  • Start with dictionary attacks: They're fastest and catch common passwords
  • Use good wordlists: rockyou.txt, SecLists, or custom lists based on target
  • Apply rules strategically: best64.rule is a good starting point
  • Monitor temperature: GPU cracking generates significant heat
  • Use sessions: For long-running attacks that might be interrupted
  • Optimize workload: Adjust -w parameter based on your system
  • Check potfile regularly: See what's been cracked already
  • Combine attack modes: Start with dictionary, then hybrid, then brute force
