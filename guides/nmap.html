What is Nmap?

Nmap (Network Mapper) is a free and open-source utility for network discovery and security auditing. It is an indispensable tool for network administrators, penetration testers, and security professionals. Nmap uses raw IP packets in novel ways to determine what hosts are available on the network, what services (application name and version) those hosts are offering, what operating systems (and OS versions) they are running, what type of packet filters/firewalls are in use, and dozens of other characteristics.

Common Commands & Examples

Here are some of the most common Nmap commands, broken down by function. The target can be a single IP, a hostname, or a network range (e.g., 192.168.1.0/24).

Basic Scans

These are the simplest scans to get started with.

# Ping Scan - Discovers which hosts are online
nmap -sn [target]
# Basic Port Scan - Scans the 1,000 most common ports
nmap [target]

Port Scanning Techniques

Control how Nmap scans for ports.

# Scan specific ports
nmap -p 80,443 [target]
# Scan a range of ports
nmap -p 1-100 [target]
# Scan all 65535 ports
nmap -p- [target]
# Fast Scan - Scans fewer ports than the default scan
nmap -F [target]

Service and Version Detection

Identify the software and version running on open ports.

# Service Version Detection
nmap -sV [target]

OS Detection

Attempt to identify the operating system of the target host.

# Operating System Detection
nmap -O [target]

Aggressive Scan (All-in-One)

An aggressive scan enables OS detection, version detection, script scanning, and traceroute.

# Aggressive Scan
nmap -A [target]

Nmap Scripting Engine (NSE)

Use Nmap's powerful scripting engine to automate tasks.

# Run default safe scripts
nmap -sC [target]
# Run a specific script (e.g., http-title)
nmap -p 80 --script http-title [target]
# Get help for a script
nmap --script-help 'http-*'

Output Formats

Save your scan results to a file.

# Save output to a text file
nmap -oN output.txt [target]
# Save output to an XML file
nmap -oX output.xml [target]
# Save in all major formats
nmap -oA output_basename [target]

Tips for Beginners

  • Be Ethical: Only scan networks and hosts that you have explicit permission to test. Unauthorized scanning is illegal.
  • Start Simple: Begin with basic scans like nmap [target] before moving to more complex options.
  • Understand the Output: Learn what 'open', 'closed', and 'filtered' mean in the context of a port scan.
  • Use -v for Verbosity: Add the -v (or -vv for more detail) flag to your commands to see more information about what Nmap is doing in real-time.
