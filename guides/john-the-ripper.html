What is John the Ripper (JtR)?

John the Ripper is a free, open-source, and highly popular password cracking tool. Its primary purpose is to perform offline attacks by taking password hashes and attempting to determine the original plaintext password. It can detect hundreds of hash and cipher types and can be run on a wide variety of platforms. It's an essential tool for penetration testers and system administrators to audit password strength.

Understanding Cracking Modes

JtR operates in several modes. Understanding them is key to using the tool effectively.

1. Wordlist Mode

This is the most common mode. JtR uses a provided list of potential passwords (a wordlist or dictionary) and hashes each one to see if it matches the target hash.

# Basic wordlist attack
john --wordlist=/path/to/wordlist.txt /path/to/hashes.txt

2. Single Crack Mode

This mode uses information from the username (e.g., 'root', 'admin') and GECOS fields to generate password guesses. It's fast and effective against simple, username-derived passwords.

# JtR's default first mode
john /path/to/hashes.txt

3. Incremental Mode (Brute-Force)

This is the 'brute-force' mode. JtR tries all possible character combinations up to a certain length. It's the most comprehensive but also the slowest mode.

# Specify the incremental mode (e.g., Alnum for alphanumeric)
john --incremental=Alnum /path/to/hashes.txt

Step 1: Prepare the Hashes

First, you need a file containing the password hashes you want to crack. For Linux systems, you might use unshadow to combine /etc/passwd and /etc/shadow.

unshadow /etc/passwd /etc/shadow > hashes.txt

Step 2: Run John

Simply point John at your hash file. It will automatically detect the hash type and cycle through its modes (Single, Wordlist, Incremental).

john hashes.txt

Step 3: Specify a Wordlist

For a more targeted attack, provide a wordlist. The RockYou wordlist is a popular choice.

# Using the popular rockyou.txt wordlist
john --wordlist=/usr/share/wordlists/rockyou.txt hashes.txt

Step 4: Show Cracked Passwords

To see the passwords that JtR has successfully cracked, use the --show flag.

john --show hashes.txt

Step 5: Specify a Hash Format

Sometimes JtR may misidentify a hash type. You can specify it manually with the --format flag.

# Example for raw MD5 hashes
john --format=raw-md5 --wordlist=wordlist.txt hashes.txt

Tips for Beginners

  • Let It Run: Password cracking can take a long time. JtR automatically saves its session, so you can stop it (Ctrl+C) and resume later by running the same command.
  • Use Good Wordlists: The success of a wordlist attack depends entirely on the quality of the wordlist.
  • Use Mangling Rules: JtR can apply rules to wordlist entries (like adding numbers or symbols) to create more password variations. This is on by default.
  • Check the Formats: Run john --list=formats to see all the hash types your build of JtR supports.
